## 运行方法

```shell
pnpm install
npm run dev
```

## 开发说明

背景图片位于public文件夹，若需要增加背景图片，按照0-7.jpg的格式更名后放在该文件夹下即可。

## 功能说明

使用css技术实现的简易的桌面图标高亮显示方案，鼠标光标在桌面甩动时，可实现图标的旋转+高亮，达到增强可见性的效果。


## 菜单说明
- 点击按钮“旋转”以后，进行甩动鼠标可以获得“图标旋转”效果
- 点击按钮“抖动”以后，进行甩动鼠标可以获得“图标抖动”效果
- 点击按钮“边框”以后，图标可以拥有边框效果
- 点击按钮“阴影”以后，图标拥有阴影效果
